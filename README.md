# Aplicación Gestor de Tareas (DESIN)

## Propósito
Esta aplicación ha sido desarrollada como examen del Tema 5 de Desarrollo de Interfaces. Implementa una interfaz gráfica con PySide6 utilizando el patrón de diseño MVC.

## Características
- Desarrollo íntegro en Python.
- Conversión automatizada a ejecutable mediante GitHub Actions.
- Generación de Artifacts para Windows y Linux.

## Instrucciones de ejecución local
1. Instalar dependencias: `pip install -r requirements.txt`
2. Ejecutar: `python main.py`
